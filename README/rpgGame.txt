## **RPG Battle Game**

### **Description**
A **turn-based RPG battle** where players choose a character class and fight an **AI-controlled enemy**.

### **Features**
- Character selection (Warrior, Mage, Archer).
- AI enemy with randomized attack power.
- Different actions: Attack, Defend, Heal, Check Stats.

### **Installation**
1. **Compile the program:**
   ```sh
   javac RPGGame.java
   ```
2. **Run the program:**
   ```sh
   java RPGGame
   ```

### **Usage**
- Choose your class and enter battle.
- Attack, defend, or heal strategically.
